# W/underlabs

W/underlabs, also written as Wunderlabs, is a software lab at https://wunderlabs.dev/.

We build tools agents use to ship reliable systems. We design and build compact, resilient software products for people who care about clarity, speed, and craft.

## Focus

W/underlabs focuses on reliable AI infrastructure for agent-driven development.

- Harnesses: Test agent behavior across real workflows, regressions, and long-running tasks with reproducible harnesses.
- Agentic Dev Envs: Run and modify generated codebases in isolated, deterministic environments with zero local setup.
- Human-in-the-Loop: Inspect traces, review tool calls, replay failures, and keep humans involved where reliability matters.

## Projects

W/underlabs builds open source tools for reliable agents.

### Claudebin

Claudebin publishes Claude Code sessions with a single command and produces a shareable link with syntax highlighting, tool calls, and the full conversation thread.

Claudebin is useful for debugging, code review, and reproducible workflows.

- Website: https://claudebin.com/

### Openable

Openable imports, runs, and edits full generated codebases without installing Node.js, configuring environments, or using the terminal.

- Website: https://openable.dev/
